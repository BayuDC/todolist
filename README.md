# Todo List - Svelte

A simple todo list app  
Created with [Svelte](https://svelte.dev)

## Development
```
# Clone this repo 
git clone https://github.com/BayuDC/todolist-svelte.git

# Change directory
cd todolist-svelte

# Install dependencies
npm install 
# or 
yarn

# Run developer server
npm run dev
# or 
yarn dev

# Open http://localhost:8080 in your browser
```
